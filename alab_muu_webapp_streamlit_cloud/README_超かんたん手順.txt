ALAB/MUU AI Web App

目的：
Macにインストールせず、ブラウザでALAB/MUUのAI判定を見るためのWebアプリです。

一番かんたんな公開方法：
Streamlit Community Cloudを使います。

手順：
1. GitHubにログイン
2. 新しいリポジトリを作る
   名前例：alab-muu-ai-webapp
3. このZIPの中身を全部アップロード
   app.py
   requirements.txt
4. https://share.streamlit.io/ を開く
5. GitHubでログイン
6. New app を押す
7. Repositoryに alab-muu-ai-webapp を選ぶ
8. Main file path に app.py と入力
9. Deploy を押す

数分待つとURLができます。
そのURLをSafariで開くだけで使えます。

毎朝の使い方：
1. 作成されたURLを開く
2. 左上の「最新データ取得」を押す
3. ALAB / MUU を選ぶ

見られるもの：
- AI判定：押し目買い候補 / 保有優位 / 様子見 / 警戒 / 危険
- 健康スコア
- 20MA / 50MA / 200MA
- 20MA割れ回数
- 20MA上連続日数
- 平均回復日数
- 上昇トレンド中の平均押し
- 52週高値更新後の平均押し
- 出来高倍率
- QQQ比RS
- SOXX比RS
- 押し目買いライン
- RR表
- 比較ランキング

注意：
これは投資判断を補助するツールです。
売買を保証するものではありません。
